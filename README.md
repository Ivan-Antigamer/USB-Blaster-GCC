USB-Blaster STM32 firmware GCC build
================================================

Port of https://github.com/eleqian/WiDSO/tree/master/MCU/USB-Blaster to GCC and "traditional" STM32F103C8T6 Bluepill board.

Altera JTAG pinout for this board
* TCK: PB3
* TDO: PB4
* TDI: PB5
* TMS: PB6
* GND to GND
